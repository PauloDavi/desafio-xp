# Front - Data XP

## Tecnologias
- Vite
- React
- Typescript
- Chakra UI
- Framer Motion
- Eslint
- Prettier

## Como executar em ambiente local

Depois de clonar o repositorio instale as dependêcias

```
yarn

// or if you use npm

npm install

```

Atualize o arquivo `.env` com a url da API deste mesmo projeto

```env
// exemplo
VITE_API_BASE_URL="<url_api>"
```

Depois execute:

```
yarn dev

// or if you use npm

npm run dev

```
